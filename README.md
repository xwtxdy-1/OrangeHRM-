# 🚀 OrangeHRM 自动化测试项目

## ✨ 项目亮点
- ✅ **智能测试生成**: 使用 AI 自动生成测试用例
- ✅ **自愈测试**: 自动修复失败的测试
- ✅ **性能监控**: 实时监控应用性能
- ✅ **安全测试**: 集成 OWASP 安全测试
- ✅ **交互式报告**: 可视化测试仪表板

## 🏆 技术栈
| 技术 | 用途 | 亮点 |
|------|------|------|
| Python + Pytest | 测试框架 | 支持并行、重试、参数化 |
| Selenium 4 | UI 自动化 | 使用最新 WebDriver 协议 |
| Playwright | 多浏览器测试 | 更快的执行速度 |
| Allure | 测试报告 | 美观的交互式报告 |
| Docker | 环境隔离 | 一致的测试环境 |
| GitHub Actions | CI/CD | 完整的 DevOps 流水线 |
